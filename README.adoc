== VagrantBoxを追加し、VMをたちあげて、プロビジョニングするまで

----
$ make setup-ytt
$ make pull-boxes
$ make install-plugin
$ vagrant up
$ make provision-development
----

==== Ansibleのプロパティやregisterなどの挙動をチェックするトライ&エラー用のMake

----
$ make provision-debug
----

== たてたVMにsshして、Dockerのバージョン確認するまで

----
$ make ssh VM_KEY=fuji-01
(VM内で)
$ docker --version
----
