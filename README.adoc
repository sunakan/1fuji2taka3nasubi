== VagrantBoxを追加し、VMをたちあげて、プロビジョニングするまで

----
$ make setup-ytt
$ make pull-boxes
$ make init-plugin
$ make up
$ make provision-provision
----

==== Ansibleのプロパティやregisterなどの挙動をチェックするトライ&エラー用のMake

----
$ make provision-hello
----

== たてたVMにsshして、Dockerのバージョン確認するまで

----
$ make ssh-fuji-01
(VM内で)
$ docker --version
----
